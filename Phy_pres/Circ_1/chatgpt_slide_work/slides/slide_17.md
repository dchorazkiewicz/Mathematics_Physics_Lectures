# 17. Basic Circuit Elements

## Goal

Introduce the basic symbols used in circuit diagrams.

## Slide layout

Title:

**Basic Circuit Elements**

Use a clean grid, preferably 2 rows × 4 columns.

Elements:

1. wire
2. battery
3. resistor
4. switch
5. ammeter
6. voltmeter
7. lamp
8. current source

Each tile contains:

* symbol
* label underneath

Bottom:

**Circuit diagrams use symbols instead of physical pictures**

## Visual details

Use consistent line thickness. Leave a lot of white space. Symbols should be large and easy to read.

## Avoid

Do not add explanations of how each element works.
