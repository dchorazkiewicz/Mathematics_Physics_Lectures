# From Situation to Formula - Projectile Motion Lecture Notes

Source package for the PDF lecture notes.

Build command:

```bash
lualatex -interaction=nonstopmode projectile_recap_notes.tex
lualatex -interaction=nonstopmode projectile_recap_notes.tex
```

The figures are generated directly in LaTeX using TikZ/PGFPlots, so there are no external image assets.
